🧾 POS System (Laravel)

📌 Description
A Point of Sale (POS) system developed using Laravel framework for managing sales, inventory, and transactions.

 ⚙️ Features
- User authentication (Admin/Cashier)
- Sales transaction processing
- Inventory management
- Sales reports
- Dashboard overview

🛠️ Technologies Used
- Laravel (PHP Framework)
- MySQL
- Blade Templates
- Bootstrap / CSS

 🚀 Installation Guide

1. Clone the repository
2. Run:
   composer install
3. Copy .env file:
   cp .env.example .env
4. Configure database in .env
5. Run migrations:
   php artisan migrate
6. Start server:
   php artisan serve
